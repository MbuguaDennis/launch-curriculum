# Curriculum

Launch is built from the ground up to serve the needs of young career starters entering technical careers. **Rriculum** is a learning repository that documents my journey through software engineering using **C# and .NET**, following a rigorous technical curriculum with a strong emphasis on professional development.

This repository serves as both a learning sandbox and a reference for concepts, exercises, and projects completed throughout the program.

---

## About

**launch-curriculum** is a personal learning repository created as part of the Launch program. It contains coursework, practice exercises, and projects focused on building a strong foundation in software engineering with **C#/.NET**.

The goal of this repository is to track progress, reinforce core software engineering concepts, and demonstrate growth toward becoming an enterprise-ready developer.
